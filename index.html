Tablas HTML

Tabla de Etiqueta

Etiqueta Nombre Descripción
<table> Table El elemento contenedor para todas las tablas HTML.
<thead> Table Head El conjunto de filas que definen los encabezados de columna en una tabla.
<tbody> Table Body El conjunto de filas que contienen datos reales de la tabla.
<tr> Table Row El contenedor de la fila de la tabla.
<td> Table Data La celda de una tabla que contiene datos
<tfoot> Table Foot El conjunto de filas que definen el pie de página en una tabla.

Atributos de Tabla

Atributo Nombre Descripción
colspan Column Span Define cuántas columnas debe abarcar un elemento td.
rowspan Row Span Define cuántas filas debe abarcar un elemento td.
